Модуль ServerBDClass.py¶

В качестве СУБД используем sqlite. Опорная схема базы данных: На стороне сервера БД содержит следующие таблицы: 1) клиент: a) логин; b) информация. 2) история клиента: a) время входа; b) ip-адрес. 3) список контактов (составляется на основании выборки всех записей с id владельца): a) id владельца; b) id клиента.
