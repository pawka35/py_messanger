Модуль server.py¶

Функции ​​сервера:​ - принимает ​с​ообщение ​к​лиента; - формирует ​​ответ ​к​лиенту; - отправляет ​​ответ ​к​лиенту; - инициализация параметров через файл ini

class server.Server¶
auth_response(auth)¶

Формирование ответа о прохождение аутентификации :param auth: флаг прохождения :return: положительный, либо отрицательный ответ

password_ask()¶

Формирование сообщения «такой клиент не найдет» :return: сформированное сообщение

presence_response(presence_message)¶

Формирование ответа при передаче служебных сообщений :param presence_message: присланное служебное сообщение :return:

read_requests(r_clients, all_clients)¶

Чтение сообщений, которые будут посылать клиенты :param r_clients: клиенты которые могут отправлять сообщения :param all_clients: все клиенты :return:

reqPassword(conn)¶

Запрос пароля у клиента (для добавление в БД) :param conn: сокет клиента :return: полученный от клиента хэш пароля

server_authenticate(con, secret_key)¶

Запрос аутентификаии клиента. сonnection - сетевое соединение (сокет); secret_key - ключ шифрования => пароль клиента, который хранится в БД (и вводит клиент у себя)

start()¶

Запуск сервера. Используем модуль select.
