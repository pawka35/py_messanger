server_log_config.py¶

Определение параметров логирования сервера

Определяем:

  1. Создаем именованный логгер с именем server

  2. Создаем обработчик с ротацией файлом по дням

  3. Определяем Форматтер сообщения

  4. Устанавливаем уровень сообщений логгера
